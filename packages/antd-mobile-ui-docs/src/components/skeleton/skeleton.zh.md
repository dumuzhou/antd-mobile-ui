## Skeleton

### 属性

| 属性     | 说明             | 类型      | 默认值  |
| -------- | ---------------- | --------- | ------- |
| animated | 是否启用动画效果 | `boolean` | `false` |

### CSS 变量

| 属性            | 说明 | 默认值 |
| --------------- | ---- | ------ |
| --width         | 宽度 | `100%` |
| --height        | 高度 | `0`    |
| --border-radius | 圆角 | `0`    |

## Skeleton.Paragraph

### 属性

| 属性      | 说明 | 类型     | 默认值 |
| --------- | ---- | -------- | ------ |
| lineCount | 行数 | `number` | `3`    |

此外，还支持 Skeleton 的 `animated` 属性。

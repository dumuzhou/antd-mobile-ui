## Skeleton

### Props

| Name     | Description                  | Type      | Default |
| -------- | ---------------------------- | --------- | ------- |
| animated | Whether to enable animation. | `boolean` | `false` |

### CSS Variables

| Name            | Description    | Default |
| --------------- | -------------- | ------- |
| --width         | Width.         | `100%`  |
| --height        | Height.        | `0`     |
| --border-radius | Border radius. | `0`     |

## Skeleton.Paragraph

### Props

| Name      | Description      | Type     | Default |
| --------- | ---------------- | -------- | ------- |

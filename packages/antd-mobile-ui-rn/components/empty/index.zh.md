# Empty 空状态

使用场景插画表示内容为空。

## 何时使用

适用于于信息为空时的占位提示。

## Empty

### 属性

| 属性        | 说明                                     | 类型            | 默认值 |
| ----------- | ---------------------------------------- | --------------- | ------ |
| description | 图片下方的描述文字                       | `ReactNode`     | -      |
| image       | 自定义图片，为 `string` 时表示图片 `URL` | `ReactNode`     | -      |
| imageStyle  | 图片样式                                 | `CSSProperties` | -      |
